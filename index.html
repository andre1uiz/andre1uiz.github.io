    My first project

    The first programming project that I made was for learning purposes. In the period of this project I was working with AI developed tools from SICK and to improve myself I started some online classes about Machine Learning using Python - Pandas.

    Because the programming language from the company was Lua and to use its machine vision products, I used the learning from the classes to apply a simple machine learning algorithm on a 2D camera using the Lua libraries available embedded on the device

    Basically, the program has the image adjustment functions from the camera and a ML classification tool, where you can add labels and train images to properly perform the inference.

    Because it was made only for study purposes, the algorithm uses the histogram from each image to learn instead of its pixels values. Due to this it is ok to assume that this program is not very robust to work on a valid application. Nevertheless, it was presented on an online fair in Brazil during the pandemic to show the ML tool from the camera.
